# pokedex

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

  App running at:
  - Local:   http://localhost:8080/ 
  - Network: http://192.168.1.103:8080/

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### Running Unit Tests

```
yarn test:unit
```
